# Data-Analysis-using-OLAP-Apache-Superset

## Description
This project demonstrates business data analysis using OLAP operations
such as slice, dice, roll-up, and drill-down. The data is prepared
for visualization using Apache Superset.

## Technologies Used
- Python (Pandas)
- OLAP Concepts
- Apache Superset

- source_code.py
